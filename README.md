# Personal Website
Página web para mostrar mis habilidades, mi experiencia laboral y la manera para contactarme.

## Template
La página esta basada del siguiente template: [link](https://htmltemplates.co/free-website-templates/black-and-green-free-bootstrap-resume-html-template).
En el que se hicieron diversos cambios al template original para acoplarlo a mis gustos.

### Cambios

- **Contenido**: Se actualizó el contenido con mis datos.
- **Imágenes**: El template original contaba con tres imágenes de los cuales solo se mantuvo el primero.
- **Contactar**: Se deshabilitó el formulario de contactar debido a que se implementará después.
- **Lenguaje**: El template está en inglés por defecto y se le agregó un botón para poder cambiar el contenido entre español e inglés. Debido a esto se agregó una transición al primer renderizado de la página para que no se vea sin contenido al visitar al comienzo.
- **Estructura**: Debido a la eliminación de imágenes se restructuró el layout de algunas de las secciones.
